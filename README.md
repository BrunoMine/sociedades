## Requitos
* Minetest 0.5.0 ou superior
* mobs
* default
* vessels
* flowers
* xpanes
* stairs
* doors
* farming
* advanced_npc
* intllib (opicional)
* hbhunger (opicional)

## Sobre
Esse modpack adiciona sociedades em minetest.
Cada sociedade possui uma caracteristica própria afim de dar certa personalidade a personagens relacionáveis.
